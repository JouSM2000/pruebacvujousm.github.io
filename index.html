Bien, hagamos un pequeño juego. Antes de pasar el mouse sobre la imagen, trata de adivinar cual es el ticker de cada accion. Un ticker o tambien conocido como etiqueta de cotización o código de cotización, es un código alfanumérico que sirve para identificar de forma abreviada las acciones de una determinada empresa que cotiza en un determinado mercado bursátil.
